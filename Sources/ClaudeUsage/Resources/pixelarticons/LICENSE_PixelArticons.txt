Used icons (24x24 viewBox SVG paths embedded in PixelIconView.swift):
  - coffee     → BadgeRegion.coffee
  - robot-face → BadgeRegion.vibe
  - clock      → BadgeRegion.cron
  - warehouse  → BadgeRegion.repo

Usage in this project: SVG paths are reproduced verbatim in source as string
constants and rendered at runtime by `PixelIconView` (Canvas + axis-aligned
rectangle parser). No binary assets are bundled.
