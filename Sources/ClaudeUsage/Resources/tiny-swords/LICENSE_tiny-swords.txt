사용 스프라이트 (Blue Units): Warrior, Lancer, Monk, Archer, Pawn
  - Mythic 3종(Warrior/Lancer/Monk)은 Resources/mythic/ 디렉토리로 분리 관리한다.
    Archer/Pawn(Legendary)만 이 tiny-swords/ 디렉토리에 남는다.
    (SwiftPM 번들은 경로를 평탄화하므로 본 LICENSE 1개로 두 디렉토리를 모두 커버한다.)
  - Archer / Pawn: Idle / Run.
  - Warrior: Idle / Run / Attack1 / Attack2 (Mythic 전용 특수 모션).
  - Lancer: Idle / Run / Attack (원본 Right_Attack, 측면 찌르기).
  - Monk: Idle / Run / Heal (Mythic 전용 특수 모션).
  - 원본 시트(192/320 정사각 셀)에서 캐릭터 bbox로 트림해 가로 strip만 커밋.
    세로 하단은 idle 발(차트 라인 정렬), 상단은 전체 모션 합집합. Lancer만 창이 길어
    투구(본체) 상단 기준으로 잘라 본체 크기를 다른 펫과 맞춘다(창 위쪽은 잘림).
  - Monk의 원본 파일명(Idle.png/Run.png/Heal.png)은 basename 유일성을 위해 Monk_ 접두사로 리네임.
