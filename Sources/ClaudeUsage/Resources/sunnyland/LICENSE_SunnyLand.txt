Included characters (idle/run strips extracted from the original
spritesheets):

- SunFox   (33x32) — player character (fox); Idle = sheet row 0 (4f),
                     Run = row 1 (6f)
- SunFrog  (35x32) — frog enemy; Idle (4f) + Run = jump cycle (3f)
- Oposum   (36x28) — oposum enemy; single walk strip (6f) for all actions
