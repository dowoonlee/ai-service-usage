Intersect Engine Free Asset Pack — Terrain Tile sprites
Source:  https://github.com/AscensionGameDev/Intersect-Assets (branch: main_full)
Folder:  resources/tilesets/Autotiles_Ground.png + Autotiles_Water.png

Each tile is a 32x32 cell extracted from the autotile sheet:

  tile_grass.png     ← Autotiles_Ground.png   (col 1, row 1)
  tile_sand.png      ← Autotiles_Ground.png   (col 7, row 0)
  tile_mountain.png  ← Autotiles_Ground.png   (col 10, row 7)
  tile_water.png     ← Autotiles_Water.png    (col 1, row 4)
  tile_lake.png      ← Autotiles_Ground.png   (col 1, row 12)
